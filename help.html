How to Use the Planner

Private / browser-based / GitHub Pages ready

Plan carefully, then keep a backup.

This planner checks curriculum progress, prerequisites, credit load, and projected GPA using the embedded SEFB programme data.

1. Choose your programme

Start from the top controls

Pick your programme, specialization track if available, English pathway, and foreign language family. Changing programme resets the active plan because each programme has a different structure.

2. Build semesters

Search, add, move

Use the Planner page to search courses inside each semester. Locked courses can still be added, but the Audit page will explain what rule is not satisfied.

3. Check audit

Requirements are grouped by category

The Audit page compares your planned credits against the official category requirements and highlights prerequisite, fail-grade, and credit-load warnings.

4. Protect your work

Export before switching devices

Your plan is saved only in the current browser. Use Profile to export a JSON backup, then import it on another device or browser.
